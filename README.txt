========================================
  FPS 2026 FOOTBALL  —  How to play
========================================

You do NOT need to install anything, and you do NOT need the internet.

HOW TO START (Windows PC):
  1. Unzip this folder somewhere easy, like your Desktop.
     (Right-click the .zip file  ->  "Extract All..."  ->  Extract.)
  2. Open the "FPS2026" folder.
  3. Double-click the file named:   index.html
  4. It opens in your web browser and you're playing. That's it.

IF WINDOWS ASKS which program to open it with:
  Choose "Google Chrome" or "Microsoft Edge" (either works great).
  If you want, you can right-click index.html -> "Open with" -> pick one.

TIPS:
  - Your saved games are kept inside that same browser on that same PC.
  - Best experience on Chrome or Edge, in a full-size window.
  - The little play-by-play strip narrates every snap. Have fun with it.

TROUBLESHOOTING:
  - If a page looks blank, make sure you double-clicked index.html
    (not one of the other files), and try Chrome or Edge.
  - Nothing here can harm your computer — it's just a web page that
    runs on your own machine.
